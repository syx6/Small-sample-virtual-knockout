# 图表目录

## 推荐优先阅读

1. `results/figures/user_facing_method_verdict.png`  
   总结当前方法效果：哪些能力好，哪些能力还不足。

2. `results/figures/user_facing_input_output_diagram.png`  
   说明方法输入什么、输出什么。

3. `results/figures/papalexi_roc_curves.png`  
   标准 ROC 曲线，说明强响应 KO 筛选能力。

4. `results/figures/papalexi_pathway_protein_correlation.png`  
   说明多模态读数是否互相支持。

5. `results/figures/norman_true_vs_pred_system_prior.png`  
   说明连续效应预测是否准确。

6. `results/figures/norman_combo_true_vs_pred_heatmap.png`  
   说明具体敲了哪些双基因组合，真实和预测分别发生什么变化。

7. `results/figures/papalexi_single_cell_virtual_ko_distributions.png`  
   展示单细胞层面虚拟 KO 分布。

8. `results/figures/norman_single_cell_virtual_combo_distributions.png`  
   展示双基因组合 KO 的单细胞分布。

## 辅助图

- `results/figures/norman_r2_improvement_system_prior.png`  
  说明系统先验相对 additive baseline 是否提升。

- `results/figures/norman_representative_combo_bars.png`  
  展示几个代表性双基因 KO 的真实变化和预测变化。

- `results/figures/norman_prior_term_hits.png`  
  展示不同组合 KO 命中了多少系统先验 term。

- `results/figures/papalexi_single_cell_state_space_pca.png`  
  展示 Papalexi 细胞在 pathway/protein 状态空间中的分布。
